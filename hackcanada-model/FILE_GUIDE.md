## 🚀 Quick Start Sequence

### Step 1: Read the Docs (~10 min)
```
README.md          → Overview of project
24HOUR_PLAN.md     → Your timeline (READ THIS FIRST)
ANNOTATION_GUIDE.md → How to label data
```

### Step 2: Setup Environment (~5 min)
```bash
python -m venv venv
venv\Scripts\activate        # Windows
# OR: source venv/bin/activate  # Mac/Linux

pip install -r requirements.txt
python setup.py              # Verify directories created
```

### Step 3: Collect & Organize Images (~5-7 hours)
- Download images from Shopify, Craigslist, etc.
- Place in `data/raw/CLEAN/`, `LIGHT_DAMAGE/`, `HEAVY_DAMAGE/`
- Rename files using convention: `[CLASS]_[SOURCE]_[DATE]_[DAMAGE]_[NUM].jpg`

### Step 4: Annotate Dataset (~5-6 hours)
- Use Label Studio or manually fill `data/sofa_returns_dataset.csv`
- For each image: condition_class, damage_types, affected_region, confidence
- QA spot-check 20% of images

### Step 5: Train Model (~1-2 hours)
```bash
cd src
python train.py
# Outputs: models/best_model_*.pth + training logs
```

### Step 6: Run Demo (~15 min)
```bash
cd src
python app.py
# Visit: http://localhost:5000
```

## 🔧 Configuration Quick Reference

Most common tweaks are in `src/config.py`:

```python
# Model training hyperparameters
NUM_EPOCHS = 30              # More epochs = better accuracy (but slower)
BATCH_SIZE = 32              # Reduce if out of memory
LEARNING_RATE = 1e-3         # Lower = more stable, higher = faster

# Data
IMG_SIZE = 224               # Image resolution
TRAIN_SPLIT = 0.70           # Train/val/test ratios
VAL_SPLIT = 0.15
TEST_SPLIT = 0.15

# Return policy
RETURN_POLICY = {
    "CLEAN": {"refund_percent": 100},
    "LIGHT_DAMAGE": {"refund_percent": 65},
    "HEAVY_DAMAGE": {"refund_percent": 0},
}

# Review thresholds
REQUIRES_HUMAN_REVIEW_THRESHOLD = 0.65  # Flag if confidence < 65%
```

## 🆘 Troubleshooting

### "Module not found" error
```bash
pip install -r requirements.txt
```

### "CUDA out of memory"
Edit `src/config.py`:
```python
BATCH_SIZE = 16  # Reduce from 32
```

### "Model not found"
Make sure you ran `python src/train.py` first.

### "Flask not running"
```bash
cd src
python app.py
```
Then visit: http://localhost:5000

### "Slow training"
Use GPU if available. Training takes:
- CPU: 45-90 min
- GPU: 10-15 min
